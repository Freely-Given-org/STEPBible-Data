FIELD DESCRIPTIONS:
==============
This datasets has multi-line records divided by "$===========". The first line is a Header with Tests that apply to each subsequent subRecord line. SubRecord lines all start with a tab. 
References are recorded in the form of Chap:Verse.Subverse, e.g. Gen.6:1.2 is the second subverse of Gen.6:1 which can also be written as Gen.6:1b. 

The columns headed "English", "Hebrew", "Latin", "Greek" and others contain the equivalent versification for Bibles in these traditions. English is determined by NRSV, Hebrew by MT, Latin by Vulgate, Greek by LXX
Additional columns ("Other") are versions which do not conform to any of these traditions. For example, "Slavonic" follows the Greek tradition except for two passages (Lev.14:55-14:57; Jos.5:14-6:27). 
Additional "undivided" columns are supplied for texts where extra text has not been divided into subverses. 
Some editions of Hebrew, Latin or Greek texts have variations which may conform exactly with another tradition (such as English); if they do not conform with any other tradition their pattern is placed in an additional column. 

Subverses occur when text is added to verses in some Bibles. The text that occurs in that verse in all Bibles is called subverse '0'. Additional verses are called subverse '1', '2' etc. 
        For example, the moving of 6 verses from Jos.8:30-35 to follow Josh.9.2 in Greek Bibles results in the text of Jos.9.2 being numbered Jos.9:2.0 and the moved  text being numbered Jos.9:1-6


TEST: 
These examine the points at which the traditions are different. They can be used to determine which tradition is being followed in a specific Bible. 
English Bibles follow mostly the Hebrew tradition, with some differences. Because so many Bibles and most commentaries follow this same set of differences, 
English standard is defined by the NRSV which is virtually identical to KJV (it divides 1Jn.14 to create the extra verse 15). 
East-European or Orthodox Bibles tend to follow the Greek traditions, and West-European or Catholic Bibles tend to follow the Latin tradition. Jewish translations tend to follow the Hebrew tradition. 
However, many Bibles employ a mixture of traditions by following English, Hebrew, Latin or Greek traditional versification for different sections of the Bible. 

Each section has at least 4 columns:  versification numbers in the Standard English, traditional Hebrew, traditional Latin and traditional Greek (and occasionally others)
A Bible is following the numbering in a specific column if the tests above that column are TRUE for that Bible. They may be true for more than one column if these have identical versification. 
* REF.1:30=Last	is TRUE if the Bible has exactly 30 verses in chapter "REF.1"
* REF.1:30=Exist	is TRUE if the Bible has a verse numbered 30 with text in it
* REF.1:30.2=Exist	is TRUE if the Bible has at least 3 subverses in REF.1:30 (the verse that appears in all Bibles is subverse 0)
* REF.1:30=NotExist	is TRUE if the Bible does not have a verse numbered 30 with text in it (and there IS text in the previous verse, if this isn't v.1 - i.e. this chapter didn't end earlier)
* REF.1:30.2=NotExist	is TRUE if the Bible has fewer than 2 subverses in REF.1:30
* REF.1:30>1:29	is TRUE if the Bible has more words in REF.1:30 than in REF.1:29 (this test is designed with enough leeway to  work for whatever language the Bible is written in)
* REF.1:30<1:29	is TRUE if the Bible has fewer words in REF.1:30 than in REF.1:29 
* REF.1:30<1:29*2	is TRUE if the Bible has fewer words in REF.1:30 than twice the number of words in REF.1:29
* REF.1.30<1:29.0	is TRUE if the Bible has fewer words in REF.1:30 than in the subverse '0' (i.e. the text which is found in all Bibles, whether or not subverses are added) 
	(subverse 0 is identical to the text found in all Bibles. Additional text that occurs in some Bibles is put into subverse 1 onwards. There are no tests for the length of those subverses because the text may not be divided into subverses)


ALIGNMENTS:
Different types of alignment are created by various situations: 

OT: 
Concatenation	occurs when an English verse merges more than one verse in another Bible, e.g. Exo.28:29
DuplicateTarget	occurs when an English verse contains only part of a verse in another Bible e.g. Gen.50:22-23
OneToOne	occurs a difference causes all subsequent verses are displaced , e.g. Num.12:16-13:33
LongVerse	refers to a section of LXX or Vulgate which is inserted into Standard versification as a series of subverses, e.g. 1Ki.6:1.0-4 (subverse "0" is the normal text of the verse, and 1-4 are additional subverses) 
LongVerseElsewhere	occurs where text has been moved into a LongVerse elsewhere, e.g. Jos.8:30-35
LongVerse/LVElsewhere	occurs where a LongVerse has been created by moving text from elsewhere, e.g. Jos.9:2 
LongVerseDuplicated	occurs where text has been duplicated in a LongVerse elsewhere, e.g. 1Ki.12:23
LongVerse/LVDuplicated 	occurs where a LongVerse has been created by moving text from elsewhere, e.g. 1Ki.12:24
LongVerse/LVExtra	occurs where a LongVerse has been created by adding text to the text, e.g. Est.1:1. These may alternately occur as extra chapters. 
TextMayBeMissing	occurs where some ancient versions do not include any words, e.g. Neh.7:68
StartDifferent	occurs where verse divisions can occur in slightly different positions, e.g. Num.26:1
[= REF]	indicates an alternate way to refer to the same verse in the same Bible


NT: 
TextMayBeMissing	occurs where some ancient manuscripts do not include any words in a verse, e.g. Mat.12:47
PassageMoved	occurs only at Rom.16:25-27 which is found at Rom.14:24-26 in some manuscripts 
Colophon	occurs at the end of epistles, which have a verse about provenance in some manuscripts, e.g. 1Co.16:24
PassageMissing	occurs at the start of passages that are missing in some manuscripts, i.e. Mrk.9-20 & Jhn.7:53-8:11 (& 1Jn.5.7b-8a which does not  alter any versification)
StartDifferent	occurs where verse divisions can occur in slightly different positions, e.g. Rev.13:1 
Concatenation	occurs when one verse merges more than one verse in another Bible, e.g. 3Jn.1:14-15]

Occasionally the differences result in a change to verse order. This is marked by an asterix in the top row, e.g."Greek *". This occurs only in Latin "Sir" and in Greek at   Exo.20:13-15; 21.16-17; 35:8-27; 37:1--38:31; 39:32-42; 40:30-31; Num.1.24-25; 10:34; 26:15-47; Jos.24:31;  1Ki.3:1; 4.20-21, 25-28; 6:1; 7:1-12, 21, 25; 9:16, 23-25; 11:5-8; 22:46-49;  Jer.31:37; 49:7-39; Ezk.7:3-5; Mal.4.6; (using English verse numbers)
